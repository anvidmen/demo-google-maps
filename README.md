# Marker Google Maps

## Instructions

To run the application, a `.env` file must be created with the following environment variable `REACT_APP_GOOGLE_MAPS_API_KEY`, this variable contains the google maps `API_KEY`.

#### Testing

In the module run:

```sh
$ npm test
```

### Run

In the module run:

```sh
$ npm start
```

### Enter

```sh
$ open http://localhost:3000/
```
